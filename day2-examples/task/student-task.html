Photography Portfolio Website Task

Task Overview

Create a responsive photography portfolio website using CSS Grid and Bootstrap. The website should showcase your photography work and provide information about your services.

Learning Objectives

  • Implement responsive layouts using CSS Grid
  • Utilize Bootstrap components effectively
  • Create interactive elements with JavaScript
  • Apply modern design principles
  • Implement responsive design patterns

Required Sections

  • Navigation Bar: Responsive menu with links to all sections
  • Hero Section: Full-width banner with a call-to-action button
  • Portfolio Grid: CSS Grid layout showcasing your work
  • About Section: Information about you and your photography
  • Services Section: List of photography services offered
  • Contact Form: Form for potential clients to reach you

Technical Requirements

  • Use CSS Grid for the portfolio layout
  • Implement Bootstrap components where appropriate
  • Make the website fully responsive
  • Include smooth scrolling navigation
  • Add form validation
  • Implement dark mode toggle

Bonus Challenges

  • Add image lightbox for portfolio items
  • Implement image lazy loading
  • Add animations on scroll
  • Create a photo filter gallery
  • Add a blog section

Design Guidelines

  • Use a consistent color scheme
  • Implement proper spacing and typography
  • Ensure good contrast for accessibility
  • Add hover effects for interactive elements
  • Use appropriate images and icons

Evaluation Criteria

  • Code organization and cleanliness
  • Responsive design implementation
  • CSS Grid usage
  • Bootstrap component integration
  • JavaScript functionality
  • Visual design and user experience

Important Notes

  • Test your website on different devices and browsers
  • Ensure all links and forms work correctly
  • Optimize images for web use
  • Include proper comments in your code
  • Follow accessibility guidelines
