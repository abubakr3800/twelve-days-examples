Firestore CRUD Operations

Key Concept: Firestore is a NoSQL document database that syncs data in realtime.

Basic Operations

// Add a document
db.collection('tasks').add({
  text: 'Learn Firebase',
  completed: false,
  createdAt: firebase.firestore.FieldValue.serverTimestamp(),
  userId: auth.currentUser?.uid
});

// Read documents (once)
db.collection('tasks').get()
  .then(querySnapshot => {
    querySnapshot.forEach(doc => {
      console.log(doc.id, '=>', doc.data());
    });
  });

// Real-time listener
db.collection('tasks')
  .where('userId', '==', auth.currentUser?.uid)
  .onSnapshot(querySnapshot => {
    const tasks = [];
    querySnapshot.forEach(doc => {
      tasks.push({ id: doc.id, ...doc.data() });
    });
    renderTasks(tasks);
  });

// Update a document
db.collection('tasks').doc(taskId).update({
  completed: true
});

// Delete a document
db.collection('tasks').doc(taskId).delete();

Authentication

Key Concept: Firebase Authentication provides backend services to authenticate users.

Email/Password Auth

// Sign up with email/password
auth.createUserWithEmailAndPassword(email, password)
  .then(userCredential => {
    console.log('User created:', userCredential.user);
  })
  .catch(error => {
    console.error('Error:', error.message);
  });

// Sign in with email/password
auth.signInWithEmailAndPassword(email, password)
  .then(userCredential => {
    console.log('User signed in:', userCredential.user);
  });

// Sign out
auth.signOut()
  .then(() => {
    console.log('User signed out');
  });

// Auth state listener
auth.onAuthStateChanged(user => {
  if (user) {
    console.log('User is signed in:', user.email);
  } else {
    console.log('User is signed out');
  }
});

Google Sign-In

// Google Sign-In
const provider = new firebase.auth.GoogleAuthProvider();

auth.signInWithPopup(provider)
  .then(result => {
    console.log('Google user:', result.user);
  })
  .catch(error => {
    console.error('Error:', error.message);
  });

function showError(message) {
  const errorElement = document.getElementById('errorAlert');
  errorElement.textContent = message;
  errorElement.style.display = 'block';
  setTimeout(() => {
    errorElement.style.display = 'none';
  }, 5000);
}

Deployment

Key Concept: Deploy your application to make it accessible to the world.

Firebase Hosting

  1. Install Firebase CLI: 
    npm install -g firebase-tools
  2. Login to Firebase: 
    firebase login
  3. Initialize your project: 
    firebase init hosting
  4. Deploy your app: 
    firebase deploy

Your app will be live at: https://your-project-id.web.app

Alternative: Netlify

  1. Drag and drop your public folder to Netlify
  2. Or connect your GitHub repository
  3. Configure build settings (if needed): 
    Build command: npm run build
Publish directory: public
  4. Deploy!
