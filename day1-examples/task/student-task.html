Day 1 - Student Task

Task Description

Create a personal portfolio page that demonstrates your understanding of HTML and CSS concepts learned today. Your portfolio should include the following sections:

Required Elements:

  1. Header Section
    • Your name as the main heading
    • A navigation menu with at least 3 links
    • Use semantic HTML tags
  2. About Me Section
    • Create a profile card with your information
    • Include a placeholder image
    • Add hover effects to the card
  3. Skills Section
    • Use flexbox to display your skills
    • Each skill should be in its own card
    • Add different colors for different skill categories
  4. Contact Form
    • Create a contact form with name, email, and message fields
    • Add proper validation for email field
    • Style the form with CSS

CSS Requirements:

  • Use the box model properly (padding, margin, border)
  • Implement flexbox for layout
  • Add hover effects and transitions
  • Use different CSS selectors (class, id, element)
  • Make the design responsive

Bonus Challenges:

  • Add a dark/light mode toggle
  • Create an animated skill progress bar
  • Add a photo gallery section
  • Implement a mobile-friendly navigation menu

Submission Guidelines:

  • Create a new HTML file named portfolio.html
  • Include all CSS in the same file (in the <style> tag)
  • Add comments to explain your code
  • Test your page in different browsers
  • Make sure all links work and forms are properly structured

Evaluation Criteria:

  • Proper use of HTML5 semantic elements
  • Clean and organized CSS code
  • Responsive design implementation
  • Visual appeal and user experience
  • Code comments and documentation

Tips:

  • Start with the HTML structure first
  • Use the browser's developer tools to debug
  • Test your page at different screen sizes
  • Don't forget to validate your HTML and CSS
  • Use placeholder images from placehold.co
